# Temporial

Temporial is the asset layer for the spatial and physical-AI era — the 2D-to-4D engine
that turns a single image into a clean, rigged, animated 3D model.

This is the company overview + pitch site. To connect, reach Andrew at https://andtay.com.

## View
Open `index.html` in a browser. The page is self-contained (3D models embedded); it needs
internet for fonts and the `model-viewer` library.

## Structure
- `index.html` — the site
- `assets/` — videos, images, 3D models, and logos
